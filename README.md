This C++ program implements a **menu-driven Doubly Linked List** using dynamic memory allocation. A custom `node` class is used to represent each element of the list, where every node stores an integer value (`data`) and two pointers (`next` and `prev`) that connect it to the next and previous nodes. The program maintains two external pointers, `head` and `tail`, to efficiently access the first and last nodes of the list, while a global variable `len_node` keeps track of the total number of nodes currently present.

Initially, the user specifies the size of the linked list and enters the values for each node. During creation, memory is allocated dynamically using the `new` operator, and each newly created node is linked to the previous node through both `next` and `prev` pointers. The first node becomes both the `head` and `tail`, and as additional nodes are added, the `tail` pointer is updated to always point to the last node in the list.

The program provides four major operations through a menu-driven interface. The **Insert** operation inserts a new node at a user-specified index. It traverses the list to locate the desired position and updates the `next` and `prev` pointers of the surrounding nodes so that the new node is correctly linked into the list. If insertion occurs at the beginning, the `head` pointer is also updated. The function validates the index and displays an error message if the specified position is outside the valid range.

The **Delete** operation removes a node based on its data value. The search begins from the `tail` and moves backward using the `prev` pointers until the required node is found. The program correctly handles all deletion cases, including deleting the only node in the list, deleting the first node, deleting the last node, and deleting a node from the middle of the list. In each case, the surrounding links are adjusted, and the `head` or `tail` pointer is updated whenever necessary to maintain the integrity of the linked list.

The **Append** operation inserts a new node at the end of the list without traversing the entire structure. Since the program maintains the `tail` pointer, the new node is directly linked after the current last node, and the `tail` pointer is updated to reference the newly appended node. This makes appending an efficient constant-time operation.

The **Extend** operation allows multiple nodes to be added to the end of the list in a single execution. The user specifies the number of elements to be inserted and enters their values one by one. Each new node is dynamically allocated, connected to the existing tail, and the `tail` pointer is updated after every insertion. This operation efficiently expands the linked list without requiring repeated menu selections.

After every operation, the program traverses and displays the linked list in both forward and reverse directions. Forward traversal begins from the `head` and follows the `next` pointers until the end of the list, while reverse traversal starts from the `tail` and follows the `prev` pointers back to the beginning. These displays verify that the bidirectional links are correctly maintained after every modification.

The program also includes several debugging statements that print the memory addresses of the `head`, `tail`, and newly created nodes. These outputs help verify pointer updates during insertion, deletion, and other operations. If all nodes are deleted, the program detects that the list has become empty and prompts the user to create a new linked list before continuing further operations.

Overall, this program demonstrates the complete implementation of a doubly linked list using object-oriented programming concepts in C++. It showcases dynamic memory allocation, pointer manipulation, bidirectional traversal, insertion at a specified position, deletion by value, appending, extending the list, and proper maintenance of `head`, `tail`, and node count. The menu-driven design makes the program interactive and serves as an effective demonstration of the fundamental operations and internal working of a doubly linked list.
